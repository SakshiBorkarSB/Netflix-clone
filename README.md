# Netflix-clone
A responsive Netflix landing page clone built using pure HTML and CSS.
This project focuses on recreating the layout and design of the official Netflix homepage for frontend practice.

---

## Features 

* Clean Netflix-inspired UI
* Responsive layout 
* Styled buttons and navigation bar
* Structured layout using Flexbox / CSS Grid

---

## Tech Stack

* HTML5
* CSS3
* Flexbox
* CSS Grid (if used)
